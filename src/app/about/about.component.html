Our Technology Stack

Abbali is built using a powerful and modern full-stack architecture to ensure performance, security, and scalability:

  • Frontend: Angular - a dynamic and responsive framework for smooth user experience across all devices.
  • Backend: ASP.NET Core - a robust and secure web API handling business logic and authentication.
  • Database: Microsoft SQL Server - managing product inventory, user data, and orders with high reliability.

This combination allows us to maintain a fast, reliable, and secure platform to serve our customers efficiently.

Our Mission
